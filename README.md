# X32-PROS
x32PROS - Operating system written in C and Assembly (NASM)

# Purpose
Make a simple operating system, or at least try

# Build instruction
- Building
        git clone https://github.com/Kross1de/x32PRos.git
        cd x32PRos
        make
- Running
        qemu-system-i386 -kernel kernel
